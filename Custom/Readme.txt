1.Постановка
  1.1 В карточку сотрудника добавить новое поле «Подразделения по совместительству» типа «Множественная подстановка» на справочник подразделений. Поле не обязательное для заполнения. Доступ на редактирование поля только у Администраторов СЭД. У остальных пользователей доступ только на просмотр (не отменяет необходимость заполнения поля «Подразделение»);
  1.2      На этапе «Подготовка»:
   b.      Если у пользователя в списке сотрудников в полях «Подразделение» и «Подразделения по совместительству» нет ни одного подразделения по той компании, которая проставилась в карточку при создании, то поле «Подразделение» в карточке остаётся пустым, и пользователь не сможет сохранить карточку, так как поле «Подразделение» будет обязательным. В выпадающем списке (или всплыв.окне) выбора подразделений будет пусто;
   c.      Если у пользователя в списке сотрудников в полях «Подразделение» и «Подразделения по совместительству» есть только одно подразделение по той компании, которая проставилась в карточку при создании, то нужно проставить это подразделение в поле «Подразделение» в карточку документа. В выпадающем списке (или всплыв.окне) выбора подразделений будет доступно только одно — это подразделение.
   d.      Если у пользователя в списке сотрудников в полях «Подразделение» и «Подразделения по совместительству» указано больше одного подразделения по той компании, которая проставилась в карточку при создании, то поле «Подразделение» в карточке остаётся пустым, и пользователь должен вручную выбрать подразделение из доступных ему подразделений. В выпадающем списке (или всплыв.окне) выбора подразделений будут отображаться все его подразделения по указанной в карточке компании.

2.Реализация и логика работы модуля
Добавлен обработчик на этап карточки «Подготовка» который по полю «Компания» для пользователя в поле «Инициатор» проверяет наличие этой компании в полях пользователя: «Подразделение» и «Подразделения по совместительству» и в случае:
1)	Для данной компании нет подразделений которые есть у пользователя – поле «Подразделение» в карточке очищается.
2)	Для данной компании у пользователя есть только одно подразделение – в поле «Подразделение» проставляется данное подразделение.
3)	Для данной компании у пользователя более одного подразделения – поле «Подразделение» в карточке очищается. 

3.	Настройка
Добавить обработчик «Обработчик на проставления подразделения в зависимости от компании пользователя» к списку.
